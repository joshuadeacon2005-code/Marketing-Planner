# TRANSFORM BRAND ASSET PAGES INTO COMPREHENSIVE BRAND PAGES WITH CALENDARS

## Objective
Convert the current brand asset pages into full-featured brand management pages, each with its own pre-filtered calendar showing that brand's complete marketing plan. All brand calendars feed into and contribute to the main unified calendar on the dashboard/home page.

---

## CURRENT STATE ANALYSIS

**Current Structure:**
- Brand pages exist primarily in the Assets section
- Each brand has an asset library view
- Navigation: Assets → Select Brand → View brand's assets

**What We're Building:**
- Full brand pages that are first-class destinations
- Each brand has its own dedicated marketing calendar
- Brand calendar = pre-filtered view of main calendar (showing only that brand's content)
- All brand data contributes to main dashboard calendar
- Brand pages become command centers for managing each brand's marketing plan

---

## ARCHITECTURE OVERVIEW

```
Main Dashboard Calendar (All Brands)
    ↓
    ├── Skip Hop Brand Page → Skip Hop Calendar (filtered)
    ├── Beaba Brand Page → Beaba Calendar (filtered)
    ├── Bloom Brand Page → Bloom Calendar (filtered)
    └── [Other Brands] → [Their Calendars] (filtered)

Each brand calendar shows:
- Social posts for that brand
- Email campaigns for that brand
- Events featuring that brand
- Promotions/discounts for that brand
- Tasks related to that brand
```

**Key Principle:** 
One source of truth (main calendar data), multiple views (dashboard shows all, brand pages show filtered subset)

## STEP 3: CREATE BRAND CALENDAR COMPONENT

### 3.1 Brand-Specific Calendar (Pre-filtered View)

**File: `client/src/components/brand-calendar.tsx`**

```typescript
import { useMemo, useState } from "react";
import { useQuery } from "@tanstack/react-query";
import { Calendar, momentLocalizer, View } from 'react-big-calendar';
import moment from 'moment';
import 'react-big-calendar/lib/css/react-big-calendar.css';
import { Button } from "@/components/ui/button";
import { Badge } from "@/components/ui/badge";
import { 
  Select, 
  SelectContent, 
  SelectItem, 
  SelectTrigger, 
  SelectValue 
} from "@/components/ui/select";
import { Checkbox } from "@/components/ui/checkbox";
import { Label } from "@/components/ui/label";

const localizer = momentLocalizer(moment);

interface BrandCalendarProps {
  brandId: string;
  brandColor: string;
}

export function BrandCalendar({ brandId, brandColor }: BrandCalendarProps) {
  const [view, setView] = useState<View>('month');
  const [date, setDate] = useState(new Date());
  
  // Sub-filters for brand calendar
  const [filters, setFilters] = useState({
    platforms: [] as string[],
    regionIds: [] as string[],
    contentTypes: ['social', 'email', 'event', 'promotion', 'task'],
  });

  // Get all regions for filter
  const { data: regions } = useQuery({
    queryKey: ["/api/regions"],
  });

  // Fetch all content FOR THIS BRAND ONLY
  const { data: socialPosts } = useQuery({
    queryKey: ["/api/social", { brandId }],
    queryFn: async () => {
      const res = await fetch(`/api/social?brandId=${brandId}`, {
        headers: { Authorization: `Bearer ${token}` }
      });
      return res.json();
    },
    enabled: filters.contentTypes.includes('social'),
  });

  const { data: emailCampaigns } = useQuery({
    queryKey: ["/api/email", { brandId }],
    queryFn: async () => {
      const res = await fetch(`/api/email?brandId=${brandId}`, {
        headers: { Authorization: `Bearer ${token}` }
      });
      return res.json();
    },
    enabled: filters.contentTypes.includes('email'),
  });

  const { data: events } = useQuery({
    queryKey: ["/api/events", { brandId }],
    queryFn: async () => {
      const res = await fetch(`/api/events?brandId=${brandId}`, {
        headers: { Authorization: `Bearer ${token}` }
      });
      return res.json();
    },
    enabled: filters.contentTypes.includes('event'),
  });

  const { data: promotions } = useQuery({
    queryKey: ["/api/promotions", { brandId }],
    queryFn: async () => {
      const res = await fetch(`/api/promotions?brandId=${brandId}`, {
        headers: { Authorization: `Bearer ${token}` }
      });
      return res.json();
    },
    enabled: filters.contentTypes.includes('promotion'),
  });

  const { data: tasks } = useQuery({
    queryKey: ["/api/tasks", { brandId }],
    queryFn: async () => {
      const res = await fetch(`/api/tasks?brandId=${brandId}`, {
        headers: { Authorization: `Bearer ${token}` }
      });
      return res.json();
    },
    enabled: filters.contentTypes.includes('task'),
  });

  // Transform all data into calendar events
  const calendarEvents = useMemo(() => {
    const events: any[] = [];

    // Apply sub-filters (platform, region)
    const applyFilters = (item: any) => {
      if (filters.platforms.length > 0 && item.platform) {
        if (!filters.platforms.includes(item.platform)) return false;
      }
      if (filters.regionIds.length > 0 && item.regionId) {
        if (!filters.regionIds.includes(item.regionId)) return false;
      }
      return true;
    };

    // Social posts
    if (socialPosts) {
      socialPosts.filter(applyFilters).forEach((post: any) => {
        events.push({
          id: `social-${post.id}`,
          title: `📱 ${post.platform} Post`,
          start: new Date(post.scheduledDate),
          end: new Date(post.scheduledDate),
          resource: { type: 'social', data: post },
          color: brandColor,
          textColor: '#ffffff',
        });
      });
    }

    // Email campaigns
    if (emailCampaigns) {
      emailCampaigns.filter(applyFilters).forEach((email: any) => {
        events.push({
          id: `email-${email.id}`,
          title: `📧 ${email.subject}`,
          start: new Date(email.scheduledDate),
          end: new Date(email.scheduledDate),
          resource: { type: 'email', data: email },
          color: brandColor,
          textColor: '#ffffff',
        });
      });
    }

    // Events
    if (events) {
      events.filter(applyFilters).forEach((event: any) => {
        events.push({
          id: `event-${event.id}`,
          title: `🎪 ${event.name}`,
          start: new Date(event.startDate),
          end: new Date(event.endDate),
          resource: { type: 'event', data: event },
          color: brandColor,
          textColor: '#ffffff',
          allDay: true,
        });
      });
    }

    // Promotions
    if (promotions) {
      promotions.filter(applyFilters).forEach((promo: any) => {
        events.push({
          id: `promo-${promo.id}`,
          title: `🎉 ${promo.name}`,
          start: new Date(promo.startDate),
          end: new Date(promo.endDate),
          resource: { type: 'promotion', data: promo },
          color: brandColor,
          textColor: '#ffffff',
          allDay: true,
        });
      });
    }

    // Tasks
    if (tasks) {
      tasks
        .filter((t: any) => t.dueDate)
        .filter(applyFilters)
        .forEach((task: any) => {
          events.push({
            id: `task-${task.id}`,
            title: `✓ ${task.title}`,
            start: new Date(task.dueDate),
            end: new Date(task.dueDate),
            resource: { type: 'task', data: task },
            color: brandColor,
            textColor: '#ffffff',
          });
        });
    }

    return events;
  }, [socialPosts, emailCampaigns, events, promotions, tasks, filters, brandColor]);

  const eventStyleGetter = (event: any) => {
    return {
      style: {
        backgroundColor: event.color,
        borderColor: event.color,
        color: event.textColor,
        borderRadius: '4px',
        border: 'none',
        fontSize: '0.875rem',
      }
    };
  };

  const handleSelectEvent = (event: any) => {
    // Navigate to the specific content item
    const { type, data } = event.resource;
    switch (type) {
      case 'social':
        window.location.href = `/social?selected=${data.id}`;
        break;
      case 'email':
        window.location.href = `/email?selected=${data.id}`;
        break;
      case 'event':
        window.location.href = `/events?selected=${data.id}`;
        break;
      case 'promotion':
        // Open promotion detail dialog
        break;
      case 'task':
        window.location.href = `/tasks?selected=${data.id}`;
        break;
    }
  };

  return (
    <div className="space-y-4">
      {/* Sub-Filters */}
      <div className="flex flex-wrap gap-4 p-4 bg-muted/50 rounded-lg">
        {/* Platform Filter */}
        <div className="flex-1 min-w-[200px]">
          <Label className="mb-2 block text-sm">Platforms</Label>
          <div className="space-y-2">
            {['INSTAGRAM', 'FACEBOOK', 'LINKEDIN', 'TWITTER', 'TIKTOK'].map((platform) => (
              <div key={platform} className="flex items-center space-x-2">
                <Checkbox
                  id={`platform-${platform}`}
                  checked={filters.platforms.includes(platform)}
                  onCheckedChange={(checked) => {
                    setFilters({
                      ...filters,
                      platforms: checked
                        ? [...filters.platforms, platform]
                        : filters.platforms.filter(p => p !== platform)
                    });
                  }}
                />
                <label
                  htmlFor={`platform-${platform}`}
                  className="text-sm font-medium leading-none peer-disabled:cursor-not-allowed peer-disabled:opacity-70"
                >
                  {platform}
                </label>
              </div>
            ))}
          </div>
        </div>

        {/* Region Filter */}
        <div className="flex-1 min-w-[200px]">
          <Label className="mb-2 block text-sm">Regions</Label>
          <div className="space-y-2">
            {regions?.map((region: any) => (
              <div key={region.id} className="flex items-center space-x-2">
                <Checkbox
                  id={`region-${region.id}`}
                  checked={filters.regionIds.includes(region.id)}
                  onCheckedChange={(checked) => {
                    setFilters({
                      ...filters,
                      regionIds: checked
                        ? [...filters.regionIds, region.id]
                        : filters.regionIds.filter(r => r !== region.id)
                    });
                  }}
                />
                <label
                  htmlFor={`region-${region.id}`}
                  className="text-sm font-medium leading-none peer-disabled:cursor-not-allowed peer-disabled:opacity-70"
                >
                  {region.name}
                </label>
              </div>
            ))}
          </div>
        </div>

        {/* Content Type Filter */}
        <div className="flex-1 min-w-[200px]">
          <Label className="mb-2 block text-sm">Content Types</Label>
          <div className="space-y-2">
            {[
              { value: 'social', label: 'Social Posts' },
              { value: 'email', label: 'Emails' },
              { value: 'event', label: 'Events' },
              { value: 'promotion', label: 'Promotions' },
              { value: 'task', label: 'Tasks' },
            ].map((type) => (
              <div key={type.value} className="flex items-center space-x-2">
                <Checkbox
                  id={`type-${type.value}`}
                  checked={filters.contentTypes.includes(type.value)}
                  onCheckedChange={(checked) => {
                    setFilters({
                      ...filters,
                      contentTypes: checked
                        ? [...filters.contentTypes, type.value]
                        : filters.contentTypes.filter(t => t !== type.value)
                    });
                  }}
                />
                <label
                  htmlFor={`type-${type.value}`}
                  className="text-sm font-medium leading-none peer-disabled:cursor-not-allowed peer-disabled:opacity-70"
                >
                  {type.label}
                </label>
              </div>
            ))}
          </div>
        </div>
      </div>

      {/* Event Count Summary */}
      <div className="flex items-center gap-2">
        <Badge variant="outline">
          {calendarEvents.length} items scheduled
        </Badge>
        {filters.platforms.length > 0 && (
          <Badge variant="secondary">
            {filters.platforms.length} platforms selected
          </Badge>
        )}
        {filters.regionIds.length > 0 && (
          <Badge variant="secondary">
            {filters.regionIds.length} regions selected
          </Badge>
        )}
      </div>

      {/* Calendar */}
      <div className="bg-white rounded-lg border">
        <Calendar
          localizer={localizer}
          events={calendarEvents}
          startAccessor="start"
          endAccessor="end"
          style={{ height: 700 }}
          view={view}
          onView={setView}
          date={date}
          onNavigate={setDate}
          eventPropGetter={eventStyleGetter}
          onSelectEvent={handleSelectEvent}
          views={['month', 'week', 'day', 'agenda']}
          popup
          tooltipAccessor={(event) => event.title}
        />
      </div>
    </div>
  );
}
```

```typescript
      return '🎪';
    case 'promotion':
      return '🎉';
    case 'task':
      return '✓';
    default:
      return '📄';
```

```typescript
// server/routes.ts
app.get("/api/brands/:id/activity", authenticateToken, async (req: any, res) => {
  try {
    const { id } = req.params;
    const limit = parseInt(req.query.limit as string) || 20;
    
    // Fetch recent items from all content types
    const [posts, emails, events, tasks] = await Promise.all([
      storage.getSocialPosts({ brandId: id }),
      storage.getEmailCampaigns({ brandId: id }),
      storage.getEvents({ brandId: id }),
      storage.getTasks({ brandId: id }),
    ]);
    
    // Combine and transform into activity items
    const activity: any[] = [];
    
    posts.slice(0, 10).forEach(post => {
      activity.push({
        id: `social-${post.id}`,
        type: 'social',
        title: `${post.platform} post created`,
        description: post.caption?.substring(0, 100),
        status: post.assetStatus,
        createdAt: post.createdAt,
      });
    });
    
    emails.slice(0, 10).forEach(email => {
      activity.push({
        id: `email-${email.id}`,
        type: 'email',
        title: `Email campaign: ${email.subject}`,
        description: email.previewText?.substring(0, 100),
        status: email.status,
        createdAt: email.createdAt,
      });
    });
    
    events.slice(0, 10).forEach(event => {
      activity.push({
        id: `event-${event.id}`,
        type: 'event',
        title: `Event: ${event.name}`,
        description: event.description?.substring(0, 100),
        status: event.status,
        createdAt: event.createdAt,
      });
    });
    
    tasks.slice(0, 10).forEach(task => {
      activity.push({
        id: `task-${task.id}`,
        type: 'task',
        title: task.title,
        description: task.description?.substring(0, 100),
        status: task.status,
        createdAt: task.createdAt,
      });
    });
    
    // Sort by creation date and limit
    activity.sort((a, b) => 
      new Date(b.createdAt).getTime() - new Date(a.createdAt).getTime()
    );
    
    res.json(activity.slice(0, limit));
  } catch (error) {
    console.error("Get brand activity error:", error);
    res.status(500).json({ message: "Internal server error" });
  }
});
```

## KEY CONCEPTS TO REMEMBER

### 1. **Single Source of Truth**
All calendar data comes from the same database tables. Brand pages simply filter this data by `brandId`.

### 2. **Pre-filtered View**
```
Main Calendar (Dashboard):  WHERE 1=1 (all brands)
Skip Hop Page Calendar:     WHERE brand_id = 'skip-hop-id'
Beaba Page Calendar:        WHERE brand_id = 'beaba-id'
```

### 3. **Calendar Synchronization**
- Changes made on brand pages affect the main calendar automatically
- Changes on main calendar appear on relevant brand pages
- No data duplication—just different views of same data

### 4. **Navigation Flow**
```
User Flow 1: Dashboard → See all brands → Filter by brand → View details
User Flow 2: Skip Hop sidebar → Skip Hop page → See Skip Hop calendar only
User Flow 3: Social page → Create post → Select brand → Post appears on that brand's calendar
```
